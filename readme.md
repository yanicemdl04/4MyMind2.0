# 🧠 For My Mind

For My Mind est une plateforme digitale dédiée au suivi et à l'amélioration du bien-être mental.  
Elle permet aux utilisateurs de mieux comprendre leurs émotions, d’adopter de bonnes habitudes mentales et d’accéder à des outils inspirés de la thérapie cognitivo-comportementale (TCC).

---

## 🚀 Objectifs

- Aider les utilisateurs à suivre leur état émotionnel au quotidien
- Offrir des outils concrets pour améliorer la santé mentale
- Proposer du contenu personnalisé (vidéos, musiques, exercices)
- Utiliser l’intelligence artificielle pour accompagner les utilisateurs

---

## ✨ Fonctionnalités

- 🤖 **Chatbot intelligent (TCC)**  
  Assistance basée sur la thérapie cognitivo-comportementale

- 📓 **Journal de bien-être**  
  Texte, images, vidéos et notes vocales

- 📊 **Suivi des émotions (Mood Tracking)**  
  Visualisation et analyse des humeurs

- 🧘 **Exercices guidés**  
  Méditation, respiration, relaxation

- 🎯 **Recommandations personnalisées**  
  Contenu adapté selon l’état émotionnel

---

## 🏗️ Architecture technique

### Frontend
- React.js (Vite)
- UI moderne, responsive et centrée utilisateur

### Backend
- Node.js (Express)
- API REST sécurisée

### Base de données
- PostgreSQL
- Prisma ORM

### Intelligence artificielle
- Python (Machine Learning & logique prédictive)
